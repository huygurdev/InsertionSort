# InsertionSort

[22,27,16,2,18,6]

<h4>Cevap 1</h4>
1.Adım --> [16,22,27,2,18,6]
2.Adım --> [2,16,22,27,18,6]
3.Adım --> [2,16,18,22,27,6]
4.Adım --> [2,6,16,18,22,27]

<h4>Cevap 2</h4>

O(N)*O(N)

O(N^2)

<h4>Cevap 3</h4>

Avarage Case = O(N^2)
Worst Case = O(N^2)
Best Case = O(N)

<h4>Cevap 4</h4>

Dizi sıralandıktan sonra Avarage Case kapsamına girer.
Dizi yukarıda verilmiş haliyle Worst Case kapsamına girer.
